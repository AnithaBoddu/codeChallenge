# Code Challenge

##  Deployment steps

- Clone this repository: git clone https://github.com/AnithaBoddu/codeChallenge.git 
- Install dependencies
    ```> npm install```
- Run node
    ``` > node server```
- Navigate to http://localhost:3000 in the browser

## Technologies used:

- AngularJS
- Node/Express
- HTML(UI)
- SCSS
